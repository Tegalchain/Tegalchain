# Contracts Workspace

This folder contains CosmWasm contracts for the Tegalchain PoR stack. Each contract is a Rust crate with a `src/` directory, `Cargo.toml` and tests.

Planned contracts:
- tXc/ (TegalTransitSynthetic Carbon)
- tXb/ (TegalTransitSynthetic Biodiversity)
- tXw/ (TegalTransitSynthetic Water)
- tXs/ (TegalTransitSynthetic Soil)
- tXi/ (TegalTransitSynthetic I-REC)
- token-factory/
- por-evidence-registry/
- poref-validation/
- impact-market/
- token-retirement/
- validator-slashing/
- reward-distribution/

See contracts/DEVELOPMENT.md for setup and build instructions.
